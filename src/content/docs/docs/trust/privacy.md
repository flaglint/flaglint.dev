FlagLint runs locally in your repository. The CLI does not upload source code, flag keys, reports, or migration results to FlagLint-owned infrastructure.

The public website has a Stay updated form for CLI release announcements and important product updates. For the hosted privacy page, see:

- [flaglint.dev/privacy](https://flaglint.dev/privacy)

## Local Reports

Reports generated by `audit`, `scan`, `migrate`, and `validate` stay wherever you write them. Treat generated reports as source-derived artifacts because they may include file paths and flag keys.
